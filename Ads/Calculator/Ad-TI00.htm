0.2 Basic Operation

This program is for engineering calculation with a TI-86 calculator.

It is supposed that you already know how to use TI-86 calculator.

You have many access to learn the basic operations for TI-86 operations:

  • TI-86 Guide Book
  • TI-86 Basic Operation by Carolyn Meitler

However, you do not have to know all the functions of TI-86 Calculator.

You need simply enter your number and press the RETURN key.

Fig. 0.3-1 Name and Edit
A. Press PRGM. The screen will be shown as Fig.0.3-1.

The NAME menu holds the programs inside the calculator for running.

The EDIT menu is for editting the program. The experienced users may revise the existing programs or created new programs.

However, EngineerMate recommends you should not edit the programs shipped with this calculator.

A small mistake may cause serious design problem and cause huge loss for a project.
