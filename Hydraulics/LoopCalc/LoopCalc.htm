Loop System Calculations

The Hardy-Cross method is widely used in analyzing simple pipe networks in civil engineering. However, sometimes there may be a need to use it in plumbing engineering though, in most cases, probability methods, such as using Hunter Curves to determine flow without balancing, are used for pipe sizing and hydraulic calculation.

EngineerMate's Canculation Tool

EngineerMate provides a simple tool for loop calculations with the following steps:

Network Layout

Firstly, a drafted system plan have to be drawn with loops and members for each of them. A loop must have three or more pipes, which are called members. Therefore, a loop must has three or more members.

Input Data

You need to input the following before the system is calculated:

  • Select Flow Unit.
  • Input Control Point Name and Pressure.
  • Input Hazen-William C (HWC) for the pipe material you use the most so that we do not have to type in one by one and save time.
  • Input Number of Loops and Members.
  • Fill in Data, such as names of members and nodes, flows, diameters,pipe lengths and fittings, etc.
